A simple, lightweight distribution

You've reached the website for Vidor OS, a lightweight and flexible Linux® distribution that tries to Keep It Simple.

What Makes Vidor OS Different

Built on Debian Bookworm: Stable foundation without the bloat.

English Base & CLI First: System boots directly to terminal, perfect for power users.

Hardware Optimized: Special support for WiFi adapters (Realtek RTL8812AU).
