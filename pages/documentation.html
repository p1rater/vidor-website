Documentation

Learn how to use and develop for Vidor OS. Our comprehensive documentation covers everything from installation to advanced configuration.

System Architecture

Base Layer: Built on Debian Bookworm for stability and compatibility.

Kernel: Standard Debian kernel with custom Realtek (RTL8812AU) and Atheros WiFi drivers via DKMS.

Boot System: Legacy boot support with optimized bootloader for maximum compatibility.

Default Environment: English language, CLI-first boot. No bloatware, only essential services.

Hardware Support

WiFi Compatibility: Built-in support for Alfa adapters and other Realtek/Atheros devices.

ThinkPad Optimized: Special tuning for ThinkPad laptops and ultrabooks.

Legacy Boot: Works seamlessly with legacy BIOS on all compatible systems.

vidorinstall Command

After booting, run vidorinstall to configure your system:

Desktop Environments: Choose between XFCE (lightweight) or GNOME (modern)

Pentest Tools: One-command installation of Aircrack-ng, GQRX, HackRF, SDR, and other security tools

Custom Packages: Install additional tools as needed

Getting Started

Installation: Boot from USB/DVD and follow the installer

First Login: Username: badboy, Terminal as default shell

Customization: System boots to terminal for maximum control and customization

Updates: Use apt update && apt upgrade to keep your system up-to-date
